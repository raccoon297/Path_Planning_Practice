
이 알고리즘은 드론이 장애물을 피해서 신호원을 찾아가는 알고리즘이다.
이떄 드론은 목표의 좌표값을 모르고, 장애물을 피해서 신호의 세기가 커지는 방향으로 움직여서 목표한 RSSI값의 위치를 찾는다.

주요 기능들과 특징은 다음과 같다.

1. 사용한 알고리즘은 PPO
2. 맵 전체 크기는 600*600인 2D로 설정
3. 드론은 1대, 출발 위치는 왼쪽 아래 (20,20)으로 고정
4. 장애물은 4개, 고정된 위치에 반지름 40인 원 형태로 표현함.
5. 목표물은 1개, 매 애피소드마다 랜덤으로 배치. 이를 통해 목표가 맵 어디에 생성되어도 찾을 수 있도록 학습함.
7. 목표물은 맵 밖에 생성 되지 않고, 장애물과 최소 10 이상의 간격을 유지함.
8. 목표는 반지름 250의 원형태로 신호를 내보내고 있음.
9. 드론은 목표하는 RSSI의 값 이상이면 목표에 도달했다고 판단함.
10. 드론이 목표에 도달하거나 벽에 부딪히거나 장애물에 부딪히면 종료.
11. 드론들은 맵 밖으로 나갈 수 없음.
12. 드론은 목표물의 좌표를 모르며, 신호의 세기, 즉 RSSI에 의해서 목표를 찾음.
13. 학습이 완료 된 후 경로를 보여주는 애니메이션 생성.
14. 목표물이 발산하는 신호의 반경은 반투명한 붉은 구역으로 나타나 있음.
15. 애니메이션 옆이나 밑에 이 학습의 대한 평가를 볼 수 있는 에피소드 별 학습상태 그래프 추가
16. 사용한 프로그램은 VScode
17. 보기 편하게 하나의 연결된 코드가 아니라 4개의 코드로 분할 작성(env, ppo, train, test)
18. env는 환경 및 리워드 설정, train에는 학습과 pth 생성. ppo는 신경망, test는 학습된 정보를 바탕으로 시뮬레이션을 1회 실행 후 에니메이션 생성을 담당함.
19. 완성된 PPO 모델이 내장 되어있는 라이브러리가 아니라 직접 ppo알고리즘을 구현하였음.
20. 라이브러리는 기본 내장된 라이브러리는 제외하고 numpy, matplotlib, pytorch, gymnasium 사용함.
